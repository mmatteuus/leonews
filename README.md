# LeoNews

Portal de notícias de Araguaína feito com Vite + React + TypeScript + Tailwind.

## Scripts

- `npm install` – instala dependências
- `npm run dev` – ambiente de desenvolvimento
- `npm run build` – build de produção
- `npm run preview` – pré-visualizar o build

## Deploy no GitHub Pages

O workflow `.github/workflows/vite-gh-pages.yml` gera o build e publica o conteúdo de `dist/`.

## Ambiente

Não é necessário `.env` para as integrações atuais (APIs públicas sem chave).
